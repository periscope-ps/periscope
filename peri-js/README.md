# peri-js

## Dependencies

### node.js (v0.10.28)

#### OSX

if homebrew

`$ brew install node`

else

`$ git clone git://github.com/ry/node.git`

`$ cd node`

`$ ./configure`

`$ make`

`$ make install`

#### Ubuntu

`$ sudo apt-get install g++ curl libssl-dev apache2-utils`

`$ sudo apt-get install git-core`

`$ git clone git://github.com/ry/node.git`

`$ cd node`

`$ ./configure`

`$ make`

`$ make install`

## Test Installation

`$ node -v`

`$ npm -v`

## Install Libs

`$ npm install`

`$ bower install`

## Start App

`$ npm start`
